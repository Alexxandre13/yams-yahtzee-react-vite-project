## Description

This is a react/Nextjs version of the Yams / Yahtzee game. You can choose the number of players, enter their name and reset the game.

The app is working.

## Start the dev environment

```bash
npm run dev
```

```bash
npm run serve
```

## Build the project

```bash
npm run build
```

Files are build into the dist folder.
